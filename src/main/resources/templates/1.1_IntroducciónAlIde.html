Introducción al IDE

Un IDE (Integrated Development Environment) es un entorno de desarrollo integrado que proporciona un conjunto completo de herramientas para facilitar el desarrollo de software. Un IDE combina varias herramientas de desarrollo en una única aplicación, proporcionando una interfaz cohesiva para escribir, depurar, compilar, y gestionar proyectos de software.

Hay una cantidad variadas de Ides pero en nuestro caso usaremos Visual Studio Code
