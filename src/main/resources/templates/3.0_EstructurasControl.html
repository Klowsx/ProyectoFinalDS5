3.0 Estructuras de Control

En Java, las estructuras de control son esenciales para dirigir el flujo de ejecución de un programa. A continuación se presentan las principales estructuras de control en Java, junto con sus definiciones:

  1. Estructuras de control condicionales: Permiten tomar decisiones dentro del programa mediante el uso de condiciones. Ejemplos incluyen if, else if, else, y switch.
  2. Estructuras de control iterativas: Permiten repetir una secuencia de instrucciones múltiples veces. Ejemplos incluyen for, while, y do-while.

Explicación

Comparadores en Java

Los comparadores se utilizan en las estructuras de control para comparar valores. A continuación se presentan los más comunes:

  • ==: Igual a
  • !=: Distinto de
  • >: Mayor que
  • >=: Mayor o igual que
  • <: Menor que
  • <=: Menor o igual que

If

La estructura if se usa para tomar decisiones en el código. Evalúa una condición y, si es verdadera, ejecuta el bloque de código correspondiente.

int edad = 18;
if (edad >= 18) {
    System.out.println("Eres mayor de edad.");
} else if (edad >= 13) {
    System.out.println("Eres un adolescente.");
} else {
    System.out.println("Eres un niño.");
}

While

El bucle while repite un bloque de código mientras la condición especificada sea verdadera. 

int contador = 0;
while (contador < 5) {
    System.out.println("Contador: " + contador);
    contador++;
}

Do-While

El bucle do-while es similar al while, pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución del bloque.

int contador = 0;
do {
    System.out.println("Contador: " + contador);
    contador++;
} while (contador < 5);

For

El bucle for se utiliza cuando se conoce de antemano el número de iteraciones. Se compone de tres partes: inicialización, condición y actualización.

for (int i = 0; i < 5; i++) {
    System.out.println("i: " + i);
}
