Métodos Simples en Programación

En programación, un método es un bloque de código diseñado para realizar una tarea específica. Los métodos permiten a los desarrolladores dividir un programa en partes más manejables y reutilizables. Un método simple es aquel que realiza una tarea clara y bien definida, facilitando así la comprensión y mantenimiento del código. La creación de métodos simples ayuda a mejorar la legibilidad del código, reducir la redundancia y hacer que el proceso de depuración sea más eficiente.

¿Qué es un Método?

Un método es una función definida dentro de una clase que realiza una tarea específica. Un método puede recibir datos (parámetros), realizar operaciones y devolver un resultado.

Llamado a Métodos

Para llamar a un método, simplemente se utiliza el nombre del método seguido de paréntesis. Si el método requiere parámetros, se pasan dentro de los paréntesis.

public class Main {
    public static void main(String[] args) {
        saludar(); // Llamada a un método sin parámetros
        int resultado = sumar(5, 3); // Llamada a un método con parámetros
        System.out.println("Resultado: " + resultado);
    }

    public static void saludar() {
        System.out.println("Hola, mundo!");
    }

    public static int sumar(int a, int b) {
        return a + b;
    }
}

Parámetros

Los parámetros son datos que se pasan a un método para que los utilice. En la declaración del método, los parámetros se especifican dentro de los paréntesis.

public static int sumar(int a, int b) {
    return a + b;
}

En este ejemplo, el método sumar recibe dos parámetros enteros a y b y devuelve la suma de ambos.

Return

La palabra clave return se utiliza para devolver un valor desde un método. El tipo de valor devuelto debe coincidir con el tipo especificado en la declaración del método.

public static int sumar(int a, int b) {
    return a + b;
}

En este ejemplo, el método sumar devuelve un valor entero que es la suma de los parámetros a y b.
